# Linux Command Cheat Sheet

## Show files

```
ls
```

Show hidden files:

```
ls -a
```

---

## Change directory

```
cd foldername
```

Example:

```
cd ~/midnight-data-lab
```

---

## Create folder

```
mkdir foldername
```

---

## Remove file

```
rm filename
```

---

## Remove folder

```
rm -r foldername
```

---

## Open file editor

```
nano filename
```

---

## Check current location

```
pwd
```
