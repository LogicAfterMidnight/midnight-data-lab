# Python Cheat Sheet

## Activate virtual environment

```
source .venv/bin/activate
```

Deactivate:

```
deactivate
```

---

## Install packages

```
pip install package_name
```

Example:

```
pip install pandas matplotlib seaborn
```

---

## Save installed packages

```
pip freeze > requirements.txt
```

---

## Run Python script

```
python python/analysis.py
```

---
