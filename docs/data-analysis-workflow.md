# Data Analysis Workflow

Typical pipeline used by data analysts.

---

## Step 1 — Collect Data

Sources:

• CSV files
• APIs
• databases
• logs

---

## Step 2 — Store Data

Example storage:

```
SQLite
PostgreSQL
CSV files
```

---

## Step 3 — Query Data

Use SQL:

```
SELECT
WHERE
GROUP BY
JOIN
```

---

## Step 4 — Analyze with Python

Use libraries:

```
pandas
numpy
matplotlib
seaborn
```

---

## Step 5 — Visualize

Examples:

• charts
• dashboards
• reports

---

## Typical toolchain

```
SQL → Python → Pandas → Visualization
```
